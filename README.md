# xplyweb

Interactive Verifiable Computation Playground powered by `power_house 0.1.0` (Rust → WASM)

## Live Demo

🚀 **Deployed to GitHub Pages!**

[https://jrochub.github.io/xplyweb](https://jrochub.github.io/xplyweb)

> The site is built with React + Vite + TypeScript + Tailwind + Rust (WASM) and automatically deployed via GitHub Actions on every push to `main`.

## Features
- Real sum-check proofs in browser
- Merkle tree visualization
- Transcript explorer
- Stats & benchmarks

Built with React + Vite + TypeScript + Tailwind + Rust WASM

## How it works
- Frontend: Vite + React + TypeScript + Tailwind
- Verifiable computation: Rust compiled to WebAssembly (WASM)
- Deployment: GitHub Actions workflow (`.github/workflows/deploy.yml`)
